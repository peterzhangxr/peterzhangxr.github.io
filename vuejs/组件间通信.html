# Vue组件间通信

# 父组件向子组件传递数据

父组件通过Props向下传递数据给子组件 注：组件中的数据共有三种形式，data, props, computed

父组件

<template>
	<Child :name="name" :age="age"/>
</template>

export default {
	name: 'Parent'
	data () {
		return {
			name: 'peterzhang',
			age: 20
		}
	}
	components: {
		Child
	}
}

子组件

<template>
	<div>name: {{ name }}</div><!--<div>name: peterzhang</div>-->
	<div>age: {{ age }}</div><!--<div>age: 20</div>-->
</template>

export default {
	name: 'Child',
	props: {
		name: String,
		age: Number
	}
}

# 子组件向父组件传递数据

子组件通过$emit向父组件传递events，并把数据传递给父组件 子组件

<template>
	<button @click="changeTitle">点击我向父组件传值</button>
</template>

export default {
	name: 'Child',
	methods: {
		changeTitle() {
			this.$emit('updateTitle', '子组件请求更新父组件的Title')
		}
	}
}

父组件

<template>
	<div>{{ title }}</div><!--<div>子组件请求更新父组件的Title</div>-->
	<Child @updateTitle="onUpdateTitle"/>
</template>

export default {
	name: 'Parent'
	data () {
		return {
			title: ''
		}
	}
	components: {
		Child
	},
	methods: {
		onUpdateTitle (title) {
			this.title = title
		}
	}
}

# v-model语法糖

父组件

<template>
	count: {{ count }}
	<Child
		v-model="count"
		<!--update:value -->
		@update:value="value => this.count = value"
		:value.sync="count"
	/>
</template>

export default {
	name: 'Parent'
	data () {
		return {
			count: 0
		}
	}
	components: {
		Child
	}
}

子组件

<template>
	<div>
		value: {{ value }}
		<button @click="updateValue">更新value</button>
	</div>
</template>

export default {
	name: 'Child',
	props: {
		value: Number,
	},
	methods: {
		updateValue () {
			this.$emit('input', 3)
			//this.$emit('update:value', 3)
		}
	}
}

# 中央事件总线 EventBus

扩展Vue原型链，通过一个空的Vue实例，作为中央事件总线，Vue.prototype.$eventBus = new Vue()，用它来触发事件和监听事件，实现父子、兄弟、跨级通信。

组件A

export default {
	name: 'A',
	mounted() {
		this.$emit('DATA-A', '从A来的数据')
	}
}

组件B

export default {
	name: 'B',
	mounted() {
		this.$eventBus.$emit('DATA-B', '从B来的数据')
	}
}

组件B

export default {
	name: 'C',
	mounted() {
		this.$eventBus.$on('DATA-A', (param) => {
			cnsole.log(param)//从A来的数据
		})
		this.$eventBus.$on('DATA-B', (param) => {
			cnsole.log(param)//从B来的数据
		})
	}
}

# Provide/Inject

  • 允许一个祖先组件向所有子孙后台注入一个依赖，不论组件层级有多深。主要解决了跨级组件通信问题;
  • provide和inject绑定不是响应式的，然而，如果你传入了一个可监听的对象，那么其对象的属性还是可响应的。

假设有两个组件，B是A的子组件

export default {
	name: 'A',
	provide: {
		name: 'peterzhang'
	}
}

export default {
	name: 'B',
	inject: [ 'name' ],
	mounted () {
		console.log(this.name) //peterzhang
	}
}

# 实现Provide/Inject响应性

  • 方法一，提供祖先组件的实例
export default {
	provide() {
		return this
	}
}
  • 方法二
export default {
	provide() {
		this.theme = Vue.observable({
			color: 'blue'
		})
		return {
			theme: this.theme
		}
	}
}

# $parent/$children 与ref

  • ref: 如果用在普通的DOM元素，返回DOM对象;如果用在组件上，返回组件实例VueComponent;
  • $parent: 获取父级组件实例
  • $children: 获取子组件实例

# $attrs/$listeners

  • $attrs: 包含了父作用域中不被prop所识别的特性绑定（class和style除外），当一个组件没有生命prop时，这里会包含父作用域的绑定(class和style除外)，并且可以通过v-bind="$attrs"传入组件内部。通常配合inheritAttrs选项一起使用。
  • $listeners: 包含了父作用域中的（不含.native修饰符）v-on事件监听器，通过v-on="$listeners"传入内部组件。

父组件

<template>
	<Child name="peterzhang" age="20" />
</template>

子组件

<template>
	<div v-bind="$attrs">
		<span>{{ name }}</span><!--<span>peterzhang</span>-->
		<span>{{ age }}</span><!--<span>20</span>-->
	</div>
</template>

export default {
	name: 'Child',
	inheritAttrs: false, // false表示过滤掉在props声明的属性,
	props: {
		name: String
	},
	mounted () {
		console.log(this.$attrs) // { age: 20} 没有name
	}
}
